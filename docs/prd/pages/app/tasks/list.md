## 1. 目的

任务列表页把**队列与历史统一成一屏列表**:全部签发 / 续签 / 吊销任务的类型、状态、关联证书与各时间点一览,支持按类型 / 状态 / 关联证书 / 时间范围筛选,并提供逐任务的重试 / 取消入口。它回答「有哪些任务、各自什么状态、哪些需要我处置」。

「队列」不是独立页,而是**按进行中态筛选**的视图(进行中 = 排队 / 执行中,DEC1)。任务据执行结果**驱动**证书状态流转,但证书状态定义与真相归 certificates,本页引用不复述(DEC4)。

## 2. 功能清单

| # | 功能 | 说明(业务口径) |
| --- | --- | --- |
| F1 任务列表查看(队列 + 历史) | 统一列表展示全部任务:任务类型(签发 / 续签 / 吊销)、关联证书(及其域名)、当前状态(排队 / 执行中 / 成功 / 失败 / 已取消)、触发方式(手动 / 自动 / 清理)、入队 / 开始 / 结束时间、结果摘要 |
| F2 按类型筛选 | 按任务类型(签发 / 续签 / 吊销)筛选 |
| F3 按状态筛选 | 按当前任务状态筛选;「队列」= 按进行中态(排队 / 执行中)筛选的视图 |
| F4 按关联证书筛选 | 按关联证书(及其域名)筛选 |
| F5 按时间范围筛选 | 按时间范围筛选 |
| F6 进入详情入口 | 逐条进入该任务的任务详情页 |
| F7 重试失败任务 | 对「失败」任务发起重试,派生一个新任务(同类型、同证书)入队;原失败任务保留于历史 |
| F8 取消任务 | 对「排队」或「执行中」任务发起取消 → 已取消;执行中取消为尽力而为 |

## 3. 页面流转

| 触发 | 去向 / 结果 | 说明 |
| --- | --- | --- |
| 某任务条目 | 任务详情页(该任务) | 查看完整信息、执行日志与重试链 |
| 重试(失败任务) | 二次确认 → 派生新任务入队(排队),原失败任务留历史 | 新任务记入重试链;驱动证书从告警态回进行中态(引用不复述) |
| 取消(排队 / 执行中) | 二次确认 → 任务转「已取消」 | 执行中取消尽力而为;证书侧回退归 certificates(引用不复述) |

> 进入本页的来源包括:主导航「任务与历史」入口、总览页「任务与历史」快速入口、总览页待处理项的「失败原因」入口(定位到某证书最近任务)、证书详情页「查看关联任务」入口。

## 4. 交互说明

- **统一列表**:进行中(排队 / 执行中)与已结束(成功 / 失败 / 已取消)同表呈现;「队列」即按进行中态筛选的视图,非独立页(DEC1)。
- **每条呈现**:任务类型、关联证书(及其域名)、当前任务状态、触发方式(手动 / 自动 / 清理)、入队 / 开始 / 结束时间、结果摘要;关联证书已删除的历史任务标注「证书已删除」(只读保留,不级联删除,DEC3)。
- **筛选**:类型 / 状态 / 关联证书 / 时间范围四类,可单独或组合使用;筛选为查看辅助,不改变任何任务状态。任务历史为只增数据,列表需具备应对历史增长的分页 / 加载能力(具体交互由设计层定)。
- **进入详情**:点击任务条目进入其任务详情页(执行日志与重试链在详情页)。
- **重试**:仅「失败」任务(签发 / 续签 / 吊销均可)提供重试;重试**派生一个新任务**(同类型、同证书,触发方式 = 手动)入队(排队),原失败任务保持「失败」于历史,新任务记入重试链;重试驱动关联证书从告警态回到进行中态(签发失败→签发中 / 续签失败→续签中,或对已回退证书重新发起吊销;引用 flows/tasks.md §4,不复述)。二次确认。
- **取消**:仅「排队」或「执行中」任务提供取消 → 「已取消」;排队任务直接取消,执行中任务取消为**尽力而为**(已提交至 CA 的在途操作可能仍生效,由 certificates 下次扫描据实校正);证书侧回退 / 落点归 certificates(引用不复述)。二次确认。
- **无批量 / 无裸建**:不做批量重试 / 批量取消;不提供手动新建任务或编辑任务参数(任务只由 certificates 触发或对失败任务重试派生,DEC5)。

## 5. 验收要点

> 业务口径,每条可判真伪。

1. 统一列表展示全部任务,进行中(排队 / 执行中)与已结束(成功 / 失败 / 已取消)同表。
2. 每条任务显示:任务类型(签发 / 续签 / 吊销)、关联证书(及其域名)、当前任务状态、触发方式(手动 / 自动 / 清理)、入队 / 开始 / 结束时间、结果摘要。
3. 关联证书已删除的历史任务标注「证书已删除」,只读保留。
4. 可按任务类型筛选。
5. 可按任务状态筛选;「队列」= 按进行中态(排队 / 执行中)筛选的视图。
6. 可按关联证书筛选。
7. 可按时间范围筛选。
8. 四类筛选可组合使用,结果同时满足所有已选条件。
9. 点击任务条目进入其任务详情页。
10. 仅「失败」任务提供重试;重试派生一个新任务(同类型、同证书)入队,原失败任务保留「失败」于历史。
11. 重试后新任务记入重试链,并驱动关联证书从告警态回到进行中态。
12. 仅「排队」或「执行中」任务提供取消;取消后任务转「已取消」。
13. 「执行中」任务取消为尽力而为(在途 CA 操作可能仍生效,由证书模块下次扫描校正)。
14. 重试与取消均需二次确认。
15. 本页不含批量重试 / 批量取消。
16. 本页不提供手动新建任务 / 编辑任务参数。
17. 本页不含任务数量统计卡片(统计归总览页)。
