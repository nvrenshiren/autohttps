## 1. 目的

ACME 账户页用于**管理 ACME 账户本体**:配置并注册账户、查看账户与状态、编辑联系邮箱、对失败账户重试、移除账户。它支持**多个账户并存**(不同 CA、或同一 CA 的生产 / 测试环境),各账户独立走账户状态机。

本页**不设定默认账户**——「哪个账户为签发默认」的设定归系统设置页,本页仅**只读呈现**哪个是默认(modules/acme.md §2.1-A2)。验证方式向导与挑战执行不在本页(归验证方式向导页)。

## 2. 功能清单

| # | 功能 | 说明(业务口径) |
| --- | --- | --- |
| F1 账户列表查看 | 展示全部已配置账户,每个含:当前状态(注册中 / 已注册 / 注册失败)、目标 CA、CA 环境(生产 / 测试)、联系邮箱;并只读标示哪个为默认 |
| F2 配置并注册账户 | 选择目标 CA、填写联系邮箱、同意服务条款,发起注册;账户进入「注册中」(AT1) |
| F3 编辑联系邮箱 | 对「已注册」账户更新联系邮箱等;业务上仍视为同一账户(§2.3 跨状态动作) |
| F4 注册失败重试 | 对「注册失败」账户修正后重试注册,回到「注册中」(AT4) |
| F5 移除账户 | 移除本地账户配置,退出账户状态机(AT5);若有证书正引用该账户签发,先提示影响并二次确认 |

> 本页**不含**:验证方式向导 / 挑战执行(归验证方式向导页)、设定默认账户(归系统设置页,本页仅只读呈现默认)、批量账户操作、DNS 厂商 API 配置(MVP 不做,DEA2)。

## 3. 页面流转

| 触发 | 去向 / 结果 | 说明 |
| --- | --- | --- |
| 配置并注册账户 | 账户进入「注册中」→ CA 确认后「已注册」/ 失败后「注册失败」 | 结果在本页账户列表刷新呈现 |
| 注册失败重试 | 账户回到「注册中」 | 可先修正邮箱等再重试 |
| 编辑联系邮箱 | 留在本页,更新账户联系邮箱 | 仍为同一账户 |
| 移除账户 | 二次确认(如被证书引用则提示影响)→ 账户退出状态机,从列表移除 | 被引用证书后续续签需改用其他账户 |
| 设定默认账户(导航) | 系统设置页(默认 ACME 账户区) | 本页只读呈现默认,设定在设置页;可提供跳转入口 |

> 进入本页的来源包括:主导航「ACME 账户」入口;系统设置页选择默认账户时若无可用账户,引导来本页添加。

## 4. 交互说明

- **多账户并存**:列表可同时存在多个账户(不同 CA、或同一 CA 的生产 / 测试环境);各账户独立走账户状态机,互不影响(DEA3)。尚无任何账户时呈现空状态并引导去配置(对应「未配置」概念初始态,flows/acme.md §2.1)。
- **配置并注册**:选择目标 CA(如 Let's Encrypt),填写联系邮箱,阅读并**同意服务条款**后方可发起注册;账户进入「注册中」;CA 确认成功 → 「已注册」(可用于签发),未成功 → 「注册失败」并显示失败原因(邮箱无效 / 未同意条款 / 网络 / CA 拒绝)。
- **编辑联系邮箱**:对「已注册」账户可更新联系邮箱等,必要时对 CA 更新账户信息,业务上仍为同一账户(不改变账户状态)。
- **注册失败重试**:对「注册失败」账户修正(如更正邮箱)后重试,账户回到「注册中」。
- **移除账户**:可对「已注册 / 注册失败」账户发起移除;移除前若存在证书正引用该账户签发,**提示影响**(这些证书后续续签需改用其他账户)并**二次确认**;确认后账户退出状态机、从列表移除。
- **默认账户只读**:本页只读呈现哪个账户为默认(默认设定归系统设置页,DEA3 / settings D1);可提供跳转设置页的入口,但不在本页设定默认。
- **查看只读**:任意状态账户均可查看其信息(§2.3 跨状态动作,全状态只读)。

## 5. 验收要点

> 业务口径,每条可判真伪。

1. 展示全部已配置 ACME 账户,每个显示:当前状态(注册中 / 已注册 / 注册失败)、目标 CA、CA 环境(生产 / 测试)、联系邮箱。
2. 尚无任何账户时呈现空状态并引导去配置账户。
3. 可配置新账户:选择目标 CA、填写联系邮箱、同意服务条款后发起注册;账户进入「注册中」。
4. 未同意服务条款不能发起注册。
5. 注册成功账户进入「已注册」(可用于签发);注册失败账户进入「注册失败」并显示失败原因。
6. 「注册失败」账户可修正后重试注册,重试后回到「注册中」。
7. 「已注册」账户可编辑联系邮箱,更新后业务上仍为同一账户。
8. 可移除「已注册 / 注册失败」账户;若有证书正引用该账户签发,移除前提示影响并二次确认。
9. 移除账户后其退出状态机,不再出现在账户列表。
10. 支持多个账户并存(不同 CA / 生产·测试环境),各账户独立管理。
11. 本页只读呈现哪个账户为默认;不在本页设定默认(默认设定归系统设置页)。
12. 本页不含验证方式向导 / 挑战执行(归验证方式向导页)。
13. 本页不含批量账户操作。
