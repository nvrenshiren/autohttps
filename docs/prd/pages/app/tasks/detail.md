## 1. 目的

任务详情页呈现**单个任务的完整信息**,并承载其**执行日志**与**重试链**:本任务是第几次尝试、由哪个前序失败任务派生、后继又派生了哪些重试任务。它是排查「某证书这次签发 / 续签 / 吊销为何失败、经历了几次尝试」的落点。

对处于可操作状态的任务,本页同样提供重试 / 取消入口(与任务列表页一致,DEC1)。任务据结果驱动证书状态,但证书状态真相归 certificates,本页引用不复述(DEC4)。

## 2. 功能清单

### 2.1 信息展示

| # | 功能 | 说明(业务口径) |
| --- | --- | --- |
| F1 完整信息展示 | 任务类型、当前状态、关联证书(及其域名与当时状态)、触发方式(手动 / 自动 / 清理)、入队 / 开始 / 结束时间、执行结果、失败原因摘要 |
| F2 执行日志查看 | 本次执行完整过程日志:验证 / 挑战步骤、与 CA 的交互、错误详情;只读 |
| F3 重试链查看 | 本任务是第几次尝试、前序失败任务与后继重试任务的引用;逐条可跳对应任务详情 |

### 2.2 操作(按当前状态可用)

| # | 操作 | 可发起状态 | 结果 |
| --- | --- | --- | --- |
| F4 重试 | 失败 | 派生一个新任务(同类型、同证书)入队;原失败任务保留「失败」(§5.2 / DT1) |
| F5 取消 | 排队 / 执行中 | 任务转「已取消」;执行中取消为尽力而为(§5.4 / DT2) |

> 本页**不含**:手动编辑任务参数(DEC5);批量操作(DEC2);任务本身状态的手工改写(状态由执行 / 联动驱动)。

## 3. 页面流转

| 触发 | 去向 / 结果 | 说明 |
| --- | --- | --- |
| 重试链中前序 / 后继任务 | 该任务的任务详情页 | 沿重试链前后查看 |
| 关联证书 | 证书详情页(该证书) | 从任务侧回到被作用证书 |
| 重试(失败任务) | 二次确认 → 派生新任务(可跳新任务详情) | 原失败任务留历史;驱动证书回进行中态(引用不复述) |
| 取消(排队 / 执行中) | 二次确认 → 任务转「已取消」 | 执行中尽力而为;证书侧回退归 certificates(引用不复述) |

> 进入本页的来源包括:任务列表页点击某任务条目;证书详情页 / 总览页跳「该证书最近任务」。

## 4. 交互说明

- **完整信息只读**:展示任务类型、当前状态、关联证书(及其域名、当时状态)、触发方式、各时间点(入队 / 开始 / 结束)、执行结果与失败原因摘要;关联证书已删除时标注「证书已删除」(DEC3)。
- **执行日志**:展示本次执行完整过程日志(验证 / 挑战步骤、与 CA 的交互、错误详情),只读,供失败排查;**私钥 / 账户密钥等敏感数据不出现在日志中**(project §7 脱敏,由技术层保障)。
- **重试链**:展示本任务是第几次尝试、由哪个前序失败任务派生而来、以及它后继派生的重试任务;前序 / 后继逐条可跳其任务详情(重试不复用旧任务,而是派生新任务,重试历史即任务序列,DT1)。
- **重试**:仅「失败」任务提供重试,派生一个新任务(同类型、同证书)入队,原失败任务保持「失败」;二次确认;驱动关联证书从告警态回进行中态(引用 flows/tasks.md §4,不复述)。
- **取消**:仅「排队 / 执行中」任务提供取消 → 「已取消」;执行中取消为**尽力而为**(在途 CA 操作可能仍生效,由 certificates 下次扫描校正);证书侧回退归 certificates(引用不复述);二次确认。
- **关联证书跳转**:提供跳该任务关联证书的证书详情页入口。

## 5. 验收要点

> 业务口径,每条可判真伪。

1. 展示单任务完整信息:任务类型、当前状态、关联证书(及其域名与当时状态)、触发方式、入队 / 开始 / 结束时间、执行结果、失败原因摘要。
2. 关联证书已删除时标注「证书已删除」。
3. 展示本次执行的完整执行日志(验证 / 挑战步骤、CA 交互、错误详情),只读。
4. 执行日志不包含私钥 / 账户密钥等敏感数据。
5. 展示重试链:本任务是第几次尝试、前序失败任务、后继重试任务。
6. 重试链中前序 / 后继任务逐条可跳转其任务详情页。
7. 仅「失败」任务提供重试;重试派生一个新任务(同类型、同证书),原失败任务保留「失败」于历史;需二次确认。
8. 仅「排队 / 执行中」任务提供取消;取消后任务转「已取消」;执行中取消为尽力而为;需二次确认。
9. 提供跳转关联证书的证书详情页入口。
10. 本页操作均针对单个任务,不做批量。
11. 本页不提供手动编辑任务参数。
