## 1. 目的

系统设置页把 autohttps 的**全部设置单页分区呈现**:运行形态配置、数据存储路径、全局续签策略、默认 ACME 账户四区同页(DEC1)。它是所有全局配置的唯一集中入口。

页面**按当前运行形态显隐**形态相关项(仅桌面 / 仅服务器);**当前运行形态与数据存储路径为只读展示**(不切换形态 SF4、路径运行期不可改 SF5 / DEC3)。本页是**续签策略的唯一定义点**,但**不判定证书状态、不执行续签**(由 certificates 消费,引用不复述)。MVP 无鉴权,故本页**无账户 / 登录 / 权限**类设置(DEC2)。

## 2. 功能清单

> 按分区列出;形态差异在项内标注「仅桌面 / 仅服务器」。

### A 运行形态配置

| # | 功能 | 说明(业务口径) |
| --- | --- | --- |
| A1 查看当前运行形态 | 只读展示本实例为桌面形态或服务器形态;据此决定形态相关项可见性;**不提供切换形态** |
| A2 开机自启开关 | **仅桌面**:设置桌面应用是否随系统登录自启并常驻托盘 |
| A3 监听地址 / 端口配置 | **仅服务器**:设置 Web UI 的监听地址与端口;设为对外可达地址时**明确提示公网暴露风险**并需确认 |

### B 数据存储路径

| # | 功能 | 说明(业务口径) |
| --- | --- | --- |
| B1 查看数据存储路径(只读) | 只读展示本地持久化数据落地的根路径;运行期不可更改、无迁移语义 |

### C 全局续签策略

| # | 功能 | 说明(业务口径) |
| --- | --- | --- |
| C1 配置续签提前天数 | 设置「距到期多少天判定为即将到期」的阈值(正整数天数);供 certificates 判定 |
| C2 自动续签开关 | 设置是否在「即将到期」时自动发起续签;关闭则仅提醒、靠手动续签 |

### D 默认 ACME 账户

| # | 功能 | 说明(业务口径) |
| --- | --- | --- |
| D1 选择默认 ACME 账户 | 从 acme 已配置账户中选一个作为签发 / 续签默认;无账户或默认被删时引导去 ACME 账户页添加 / 显式选择 |

> 本页**不含**:账户 / 登录 / 权限设置(MVP 无鉴权,DEC2);通知渠道配置(MVP 仅总览页红点);主题 / 语言 / 日志级别 / 更新检查等未明示项;续签重试次数 / 间隔与扫描周期配置(SF2 / SF3);运行形态切换与形态间数据迁移(SF4)。

## 3. 页面流转

| 触发 | 去向 / 结果 | 说明 |
| --- | --- | --- |
| 修改续签提前天数 / 自动续签开关 | 保存 → 对 certificates 后续「即将到期」判定与自动续签生效 | 本页不判定证书状态 |
| 选择默认 ACME 账户 | 从 acme 已配置账户中选 → 保存默认指向 | 候选来源 acme,本页只存默认指向 |
| 无 ACME 账户 / 默认被删 | 引导去 ACME 账户页添加 / 显式选择 | 不阻塞 acme 自身账户管理 |
| 监听地址设为对外可达(仅服务器) | 提示公网暴露风险 → 确认后保存 | 等同把全部能力开放给任何可访问者(roles §3) |

> 进入本页的来源包括:主导航「设置」入口、ACME 账户页「设定默认账户」跳转。

## 4. 交互说明

- **单页分区**:运行形态 / 数据存储路径 / 续签策略 / 默认 ACME 账户四区同页呈现(DEC1)。
- **按运行形态显隐**:桌面形态显示「开机自启」、隐藏「监听地址 / 端口」;服务器形态显示「监听地址 / 端口」、隐藏「开机自启」;依据「当前运行形态」决定(flows/settings.md §5)。
- **当前运行形态(只读)**:展示本实例为桌面 / 服务器形态,让使用者明确当前能力集;**不提供切换**——运行形态是部署既定事实(两形态各自独立部署实例,SF4)。
- **开机自启(仅桌面)**:开 / 关,决定桌面应用是否随系统登录自启并常驻托盘。
- **监听地址 / 端口(仅服务器)**:设置 Web UI 监听地址与端口;默认仅监听本机 / 可信内网;设为**对外可达地址**时**明确提示公网暴露风险**(等同把全部能力开放给任何可访问者,需在前置反向代理自行加鉴权,roles §3),需使用者确认。
- **数据存储路径(只读)**:只读展示本地持久化数据(域名 / 证书 / 私钥 / ACME 账户密钥 / 根 CA 材料 / 任务历史)落地根路径;路径在部署 / 首次启动时确定,**运行期不可更改、无迁移语义**(已裁决 DEC3 / SF5)。
- **续签提前天数**:设置「距到期多少天判定为即将到期」的阈值(正整数天数);供 certificates 判定「即将到期」、驱动自动续签与红点(引用 certificates flow,不复述状态流转)。
- **自动续签开关**:决定「即将到期」证书是否自动发起续签;关闭则仅提醒、靠手动续签;开启时 certificates 亦会在后续周期扫描对「续签失败且未过期」证书自动再尝试(引用不复述)。
- **默认 ACME 账户**:从 acme 已配置账户中选一个作为签发 / 续签默认;候选清单来源 acme,本页只存「默认指向」不复制账户明细(DEC4);未配置任何账户或默认账户被删除时,引导使用者去 ACME 账户页添加 / 显式选择(不阻塞 acme 自身账户管理)。
- **保存生效**:配置保存后对后续读取该配置的模块行为生效;涉及运行载体的项(监听地址 / 端口、开机自启)的具体生效时机(即时 / 需重启守护进程 / 需重登录桌面会话)由技术层确定,本层不作机制承诺(flows/settings.md §2)。

## 5. 验收要点

> 业务口径,每条可判真伪。

1. 单页分区呈现四类设置:运行形态、数据存储路径、续签策略、默认 ACME 账户。
2. 桌面形态显示「开机自启」且不显示「监听地址 / 端口」。
3. 服务器形态显示「监听地址 / 端口」且不显示「开机自启」。
4. 当前运行形态只读展示,不提供切换。
5. 数据存储路径只读展示,运行期不可更改。
6. 可配置续签提前天数(正整数天数)。
7. 可开关自动续签;关闭时仅提醒、靠手动续签。
8. 可从 ACME 已配置账户中选择一个作为默认 ACME 账户。
9. 尚无任何 ACME 账户或默认账户被删除时,引导去 ACME 账户页添加 / 显式选择。
10. 服务器形态将监听地址设为对外可达时,明确提示公网暴露风险并需确认。
11. 续签提前天数与自动续签开关保存后,供证书模块判定「即将到期」与自动续签(本页不判定证书状态、不执行续签)。
12. 默认 ACME 账户本页只存默认指向,不复制账户明细(账户本体归 ACME 账户页)。
13. 本页不含账户 / 登录 / 权限设置。
14. 本页不含通知渠道、主题 / 语言 / 日志级别等未明示配置项。
15. 本页不含续签重试次数 / 间隔、扫描周期配置。
16. 本页不提供运行形态切换与形态间数据迁移。
