---

## 1. 目的

根 CA 列表页是自签根 CA 管理的入口页:把**全部根 CA** 的当前状态一屏总览(支持**多根 CA 并存**),并提供创建 / 导入入口、逐个导出与进入详情的操作入口。它回答「我现在有哪些根 CA、各自是否仍有效」。

内网证书的列表 / 详情 / 管理**归 certificates**,不在本页(DEC1)。

---

## 2. 功能清单

| # | 功能 | 说明(业务口径) |
| --- | --- | --- |
| F1 根 CA 列表查看 | 展示全部根 CA,每条含:名称 / 标识、当前状态(有效 / 已过期)、有效期(生效 / 失效时间)、创建方式(新建 / 导入)、指纹或序列号 |
| F2 新增入口 | 提供「创建 / 导入根 CA」入口,跳转新增根 CA 页 |
| F3 逐个导出根 CA 证书 | 每条提供导出根 CA 证书(公开部分)入口,供客户端导入信任库;只读,不改状态 |
| F4 进入详情入口 | 逐条进入该根 CA 的根 CA 详情页 |

> 本页**不含**:内网证书列表 / 管理(归 certificates,DEC1);批量创建 / 批量导出(需求未明示);显式移除 / 删除根 CA(MVP 已裁决不提供,LC5);根 CA 数量统计卡片。

---

## 3. 页面流转

| 触发 | 去向 / 结果 | 说明 |
| --- | --- | --- |
| 某根 CA 条目 | 根 CA 详情页(该根 CA) | 查看完整信息、导出、看其签发的内网证书概览 |
| 新增(创建 / 导入)入口 | 新增根 CA 页 | 创建新根 CA 或导入已有根 CA |
| 逐个导出 | 导出根 CA 证书(桌面保存本地 / 服务器浏览器下载) | 供客户端信任;留在本页 |

> 进入本页的来源包括:主导航「根 CA」入口。

---

## 4. 交互说明

- **列表范围**:列出全部根 CA(有效 / 已过期两态);尚无任何根 CA 时呈现空状态并引导去创建 / 导入(对应「尚无根 CA」的进入前空条件,flows/local-ca.md §2.1)。
- **每条呈现**:名称 / 标识、当前状态(有效 / 已过期)、有效期(生效 / 失效时间)、创建方式(新建 / 导入)、指纹或序列号。
- **状态标示**:有效 / 已过期两态(引用 flows/local-ca.md §2,不复述);「已过期」根 CA 明确标示——不可再用于签发新内网证书,提示需创建 / 导入新根 CA 接替。
- **新增入口**:提供创建 / 导入入口,跳转新增根 CA 页(两种方式在该页选择)。
- **逐个导出**:每条提供导出根 CA 证书(公开部分)入口,供使用者在客户端导入信任库(信任根 CA,glossary §3);有效 / 已过期均可导出;导出形态差异见 flows/local-ca.md §3.3——桌面形态保存到本地路径(仅桌面)、服务器形态为浏览器下载(仅服务器);只读,不改变状态。
- **进入详情**:点击根 CA 条目进入其根 CA 详情页(单根 CA 完整信息与由它签发的内网证书概览在详情页)。
- **无批量 / 无移除**:不做批量创建 / 批量导出;MVP 不提供显式移除根 CA(生命周期靠过期后创建 / 导入新根接替,LC5)。

---

## 5. 验收要点

> 业务口径,每条可判真伪。

1. 列表展示全部根 CA,每条显示:名称 / 标识、当前状态(有效 / 已过期)、有效期(生效 / 失效时间)、创建方式(新建 / 导入)、指纹或序列号。
2. 尚无任何根 CA 时呈现空状态并引导去创建 / 导入。
3. 根 CA 状态只有「有效」「已过期」两种(引用状态机,不复述)。
4. 「已过期」根 CA 明确标示,并提示需创建 / 导入新根 CA 接替。
5. 点击某根 CA 条目进入其根 CA 详情页。
6. 提供创建 / 导入入口,跳转新增根 CA 页。
7. 每条提供逐个导出根 CA 证书(公开部分)入口。
8. 桌面形态导出为保存到本地路径(仅桌面);服务器形态导出为浏览器下载(仅服务器)。
9. 支持多个根 CA 并存,统一在本列表管理。
10. 本页不含批量创建 / 批量导出操作。
11. 本页不提供显式移除 / 删除根 CA(MVP 已裁决不提供)。
12. 本页不含内网证书的列表 / 管理(归证书模块)。
