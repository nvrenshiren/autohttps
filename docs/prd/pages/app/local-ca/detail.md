## 1. 目的

根 CA 详情页呈现**单个根 CA 的完整信息**,并作为其可用操作的落点:**导出根 CA 证书**(公开部分,供客户端信任)、查看**由它签发的内网证书概览**。它是了解「这个根 CA 是什么、还有效吗、签发过哪些内网证书」的页面。

**MVP 不提供显式移除根 CA**(已裁决 LC5),生命周期靠过期后创建 / 导入新根接替;**导出仅公开根 CA 证书、不含私钥导出**(已裁决 DEC4 / LC4)。内网证书的状态 / 续签 / 到期 / 吊销 / 导出**归 certificates**,本页仅概览 + 跳转。

---

## 2. 功能清单

### 2.1 信息展示

| # | 功能 | 说明(业务口径) |
| --- | --- | --- |
| F1 完整信息展示 | 名称 / 标识、当前状态(有效 / 已过期)、有效期(生效 / 失效时间)、创建方式(新建 / 导入)、指纹或序列号、创建时间 |
| F2 内网证书概览 | 由该根 CA 签发的内网证书概览(关联域名、证书当前状态、有效期);逐条可跳该内网证书的证书详情页 |

### 2.2 操作

| # | 操作 | 可发起状态 | 结果 |
| --- | --- | --- | --- |
| F3 导出根 CA 证书 | 有效 / 已过期 | 导出根 CA 证书(公开部分)供客户端导入信任库;只读,不改状态(§2.4 / §3.3) |

> 本页**不含**:显式移除 / 删除根 CA(MVP 已裁决不提供,LC5);含私钥 / 迁移导出(MVP 已裁决不开放,DEC4 / LC4);根 CA 续期 / 自动重建(LC3);内网证书的状态管理 / 续签 / 吊销 / 导出(归 certificates,DEC1)。

## 3. 页面流转

| 触发 | 去向 / 结果 | 说明 |
| --- | --- | --- |
| 导出根 CA 证书 | 交付根 CA 证书(桌面保存本地 / 服务器浏览器下载),留在本页 | 供客户端信任;有效 / 已过期均可 |
| 内网证书概览(逐条) | 证书详情页(该内网证书) | 内网证书的完整管理在证书模块 |
| 「已过期」处置引导 | 新增根 CA 页 | 创建 / 导入新根 CA 接替签发 |

> 进入本页的来源包括:根 CA 列表页点击某根 CA 条目。

---

## 4. 交互说明

- **信息只读展示**:有效 / 已过期根 CA 均可查看完整详情(§2.4 查看详情适用两态)。
- **导出根 CA 证书**:导出**公开部分**(根 CA 证书),供使用者在目标客户端 / 设备导入信任库,完成信任根 CA(glossary §3);有效 / 已过期均可导出(已过期可导出以排查);只读,不改变状态。**仅导出公开根 CA 证书,不含私钥导出**;MVP 不开放含私钥 / 迁移导出(DEC4 / LC4)。形态差异:桌面形态保存到本地路径(仅桌面)、服务器形态为浏览器下载(仅服务器)。
- **内网证书概览**:列出由该根 CA 签发的内网证书概览(关联域名、证书当前状态、有效期);逐条可跳该内网证书的证书详情页做完整管理;内网证书的状态 / 续签 / 到期 / 吊销 / 导出归 certificates,本页不复述、不在此处执行(DEC1)。
- **已过期处置**:「已过期」根 CA 不可再用于签发新内网证书;本页提示使用者创建 / 导入新根 CA 接替(可跳新增根 CA 页);其此前签出的内网证书各自随**自身有效期**由证书状态机管理(与根 CA 到期是两件事,flows/local-ca.md §3.6)。
- **无移除**:本页不提供显式移除 / 删除根 CA(MVP 已裁决,LC5)。

---

## 5. 验收要点

> 业务口径,每条可判真伪。

1. 展示单根 CA 完整信息:名称 / 标识、当前状态(有效 / 已过期)、有效期(生效 / 失效时间)、创建方式(新建 / 导入)、指纹或序列号、创建时间。
2. 有效与已过期根 CA 均可打开详情查看(信息只读完整)。
3. 提供导出根 CA 证书(公开部分)供客户端信任;有效 / 已过期均可导出。
4. 导出仅为公开根 CA 证书,不含私钥导出;本页不提供含私钥 / 迁移导出。
5. 桌面形态导出为保存到本地路径(仅桌面);服务器形态导出为浏览器下载(仅服务器)。
6. 导出为只读动作,不改变根 CA 状态。
7. 展示由该根 CA 签发的内网证书概览(关联域名、证书状态、有效期)。
8. 内网证书概览逐条可跳转该内网证书的证书详情页;内网证书的状态 / 续签 / 吊销 / 导出在证书模块管理,不在本页。
9. 「已过期」根 CA 提示需创建 / 导入新根 CA 接替。
10. 本页不提供显式移除 / 删除根 CA(MVP 已裁决不提供)。
11. 本页不提供根 CA 的续期 / 自动重建(MVP 不做)。
