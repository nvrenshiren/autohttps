---

## 1. 目的

证书列表页是证书管理的入口页:把**全部证书**的当前状态一屏总览,支持按状态 / 签发方式 / 域名关键字筛选,并提供进入单证书详情、发起首次签发的操作入口。它回答「我现在有哪些证书、各自处于什么状态」。

---

## 2. 功能清单

| # | 功能 | 说明(业务口径) |
| --- | --- | --- |
| F1 证书列表查看 | 展示全部证书,每条含当前状态、关联域名、有效期、签发方式(ACME / 自签) |
| F2 状态筛选 | 按证书当前状态筛选查看(状态名引用 flows/certificates.md §2.1) |
| F3 签发方式筛选 | 按签发方式(公共 ACME / 自签根 CA)筛选查看 |
| F4 域名关键字筛选 | 按域名 hostname 关键字筛选查看 |
| F5 进入详情入口 | 逐条进入该证书的证书详情页 |
| F6 发起签发入口 | 跳转签发证书页,为域名发起首次签发 |

> 本页**不含**:批量签发 / 批量吊销 / 批量删除(modules/certificates.md DEC2);证书数量统计卡片(总数 / 即将到期数统计归 dashboard);单证书的续签 / 吊销 / 删除 / 导出动作(归证书详情页,DEC3)。

---

## 3. 页面流转

| 触发 | 去向 | 目的 |
| --- | --- | --- |
| 某证书条目 | 证书详情页(该证书) | 查看完整信息并执行生命周期操作 |
| 发起签发 | 签发证书页 | 为域名发起首次签发 |

> 进入本页的来源包括:总览页「查看全部证书」快速入口、总览页待处理项(定位到具体证书时可直接进详情)。

---

## 4. 交互说明

- **列表范围**:列出全部**未删除**证书(含各状态,含终态「已吊销」);已删除证书退出状态机,不出现在列表。
- **每条呈现**:关联域名(一证多域 SAN 时展示其全部关联域名,含至多一个通配符)、当前状态(引用状态机状态中文名)、有效期(生效 / 失效时间)、签发方式(ACME / 自签)。
- **筛选**:状态 / 签发方式 / 域名关键字三类筛选条件,可单独或组合使用;筛选为查看辅助,不改变任何证书状态。
- **进入详情**:点击证书条目进入其证书详情页;所有单证书生命周期动作在详情页执行。
- **发起签发**:提供发起首次签发入口,跳转签发证书页。
- **无批量与统计**:本页不做批量操作、不放统计卡片(与 dashboard 职责分离)。

---

## 5. 验收要点

> 业务口径,每条可判真伪。

1. 列表展示全部**未删除**证书,含「已吊销」证书;已删除的证书不出现在列表。
2. 每条证书显示:关联域名、当前状态、有效期、签发方式(ACME / 自签)。
3. 一证多域(SAN)的证书展示其全部关联域名。
4. 可按证书状态筛选,筛选结果只含处于该状态的证书。
5. 可按签发方式(公共 ACME / 自签根 CA)筛选,结果只含对应签发方式的证书。
6. 可按域名 hostname 关键字筛选,结果只含 hostname 命中的证书。
7. 三类筛选条件可组合使用,结果同时满足所有已选条件。
8. 点击某证书条目进入其证书详情页。
9. 提供发起签发入口,点击进入签发证书页。
10. 本页不含批量签发 / 批量吊销 / 批量删除操作。
11. 本页不含证书数量统计卡片(统计归总览页)。
