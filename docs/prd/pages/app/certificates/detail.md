## 1. 目的

证书详情页呈现**单张证书的完整信息**,并作为其**生命周期操作**的落点:续签 / 重试 / 吊销 / 删除 / 导出 / 重新签发都在此发起。各操作是否可发起严格由证书**当前状态**决定(引用 flows/certificates.md §2.3 权威转移表 / §2.4 跨状态动作),页面据此动态呈现可用 / 禁用。

## 2. 功能清单

### 2.1 信息展示

| # | 功能 | 说明(业务口径) |
| --- | --- | --- |
| F1 完整信息展示 | 关联域名、当前状态、有效期(生效 / 失效时间)、签发方式与来源(经哪个 ACME 账户 / 哪个根 CA)、证书链概要 |

### 2.2 生命周期操作(按当前状态动态可用)

> 「可发起状态」严格引用 flows/certificates.md §2.3 权威转移表与 §2.4 跨状态动作;下表只锚定业务规则,不复述状态定义。

| # | 操作 | 可发起的证书状态 | 结果 |
| --- | --- | --- | --- |
| F2 续签 / 重试续签 | 有效 / 即将到期 / 续签失败 / 已过期 | 转「续签中」(T7 / T9 / T14 / T17);续签失败态下按钮语义为「重试续签」 |
| F3 重试首签 | 签发失败 | 转「签发中」(T5) |
| F4 吊销 | 有效 / 即将到期 / 续签失败 | 转「吊销中」(T8 / T11 / T16);**已过期不可吊销**(见 §5 脚注) |
| F5 重新签发 | 已吊销 | 转「续签中」,为同域名重新获取证书(换新私钥,T20) |
| F6 删除 | 待签发 / 签发失败 / 有效 / 即将到期 / 续签失败 / 已过期 / 已吊销 | 移除条目与本地文件,退出状态机;二次确认 |
| F7 导出 | 有效 / 即将到期 / 续签中 / 续签失败 / 已过期 / 已吊销(仅本地已有文件) | 导出叶子证书 / 私钥 / 证书链,不改变状态 |
| F8 查看关联任务 / 失败原因 | 进行中态(签发中 / 续签中 / 吊销中)与失败态(签发失败 / 续签失败) | 跳任务查看进度 / 失败原因 |

> 本页**不含**:批量操作、证书自动部署(导出后由使用者自行部署,DEC1);任务的取消动作本身在任务模块执行(本页仅提供跳转入口)。

## 3. 页面流转

| 触发 | 去向 / 结果 | 说明 |
| --- | --- | --- |
| 续签 / 重试续签 | 证书转「续签中」,留在本页(可刷新看进度)或跳关联任务 | 结果成功 → 有效;失败 → 续签失败 |
| 重试首签 | 证书转「签发中」 | 结果成功 → 有效;失败 → 签发失败 |
| 吊销 | 二次确认 → 证书转「吊销中」 | 结果成功 → 已吊销;失败 → 回退吊销前状态(T19) |
| 重新签发(已吊销) | 证书转「续签中」 | 为同域名换新私钥重新获取 |
| 删除 | 二次确认 → 移除 → 返回证书列表页 | 该证书不再出现在列表 |
| 导出 | 打开导出面板 → 交付文件(留在本页) | 桌面保存本地路径 / 服务器浏览器下载 |
| 查看关联任务 / 失败原因 | 任务与历史(该证书最近任务) | 查看执行进度 / 失败日志 |
| 关联域名(逐个) | 域名详情页 | 从证书侧查看其覆盖域名 |

## 4. 交互说明

- **信息只读展示**:任意状态的证书均可查看完整详情(§2.4 查看详情适用全部状态)。
- **操作按状态动态可用**:F2–F8 各操作按证书当前状态可用 / 禁用,依据 §2.3 权威转移表——例如「已过期」证书可续签 / 删除 / 导出,但**不可吊销**;「进行中态」不可删除。
- **续签 / 重试**:续签失败态下操作语义为「重试续签」;签发失败态下为「重试首签」。发起后证书进入对应进行中态,失败原因日志在任务模块查看。
- **吊销**:仅「有效 / 即将到期 / 续签失败」可发起(如私钥泄露);二次确认;吊销失败回退吊销前状态并提示错误(证书本身未变,DC3)。
- **删除**:仅非进行中态可发起,二次确认;进行中态(签发中 / 续签中 / 吊销中)禁用删除并提示「须先在任务处取消或等任务结束」,提供跳转关联任务的入口。
- **导出**:仅「本地已有证书文件」的证书可用(有效 / 即将到期 / 续签中 / 续签失败 / 已过期 / 已吊销);待签发 / 签发中 / 签发失败无文件不可导出。导出内容可选叶子证书 / 私钥 / 证书链;**导出私钥前给出敏感数据风险提示**(project §7)。形态差异:
  - 桌面形态(仅桌面):保存到本地路径。
  - 服务器形态(仅服务器):浏览器下载。
  - 具体文件格式与打包由设计 / 技术层确定,本页只约定可选导出内容与风险提示。
- **重新签发**:已吊销证书提供「重新签发」,为同域名重新获取(换新私钥)。
- **进行中态与失败态**:提供「查看关联任务 / 失败原因」入口,跳任务模块看进度或日志。

## 5. 验收要点

> 业务口径,每条可判真伪。

1. 展示单证书完整信息:关联域名、当前状态、有效期(生效 / 失效时间)、签发方式与来源、证书链概要。
2. 任意状态的证书均可打开详情查看(信息只读完整)。
3. 续签操作仅在「有效 / 即将到期 / 续签失败 / 已过期」状态可发起;其余状态不提供。
4. 吊销操作仅在「有效 / 即将到期 / 续签失败」状态可发起;「已过期」状态**不提供**吊销。
5. 签发失败证书提供「重试首签」;续签失败证书提供「重试续签」。
6. 已吊销证书提供「重新签发」(为同域名重新获取,换新私钥)。
7. 删除仅在非进行中态可发起且需二次确认;签发中 / 续签中 / 吊销中不可删除,并提示须先在任务处取消或等任务结束。
8. 导出仅对本地已有证书文件的证书可用(有效 / 即将到期 / 续签中 / 续签失败 / 已过期 / 已吊销);待签发 / 签发中 / 签发失败不提供导出。
9. 导出内容可选叶子证书 / 私钥 / 证书链;导出私钥前有敏感数据风险提示。
10. 桌面形态导出为保存到本地路径(仅桌面);服务器形态导出为浏览器下载(仅服务器)。
11. 吊销发起需二次确认;吊销失败后证书回退到吊销前状态并提示错误。
12. 进行中态与失败态提供查看关联任务(进度 / 失败原因)的入口。
13. 删除成功后返回证书列表页,该证书不再出现。

> **脚注(吊销可发起态)**:flows/certificates.md §3.5 正文一处将「已过期」列入可吊销状态,但 §2.3 权威转移表(T8 / T11 / T16)与 §2.2 状态图均未定义「已过期 → 吊销中」。本页以**权威表 §2.3 为准**:吊销可发起态 = 有效 / 即将到期 / 续签失败,不含已过期。已过期证书的处置路径为续签(T17)或删除。
