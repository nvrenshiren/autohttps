## 1. 目的

域名列表页把**全部域名**一屏总览:每个域名的 hostname、是否通配符、分组、备注摘要,以及其**证书态投影**(有效 / 即将到期 / 失败 / 无证书)与关联证书数;支持筛选,并提供新增域名、删除域名与进入域名详情的入口。它回答「我管了哪些域名、各自的证书覆盖到什么程度」。

---

## 2. 功能清单

| # | 功能 | 说明(业务口径) |
| --- | --- | --- |
| F1 域名列表查看 | 展示全部域名,每条含 hostname、是否通配符、分组、备注摘要、证书态投影、关联证书数 |
| F2 筛选 | 按分组、证书态、hostname 关键字筛选查看 |
| F3 新增域名 | 录入 hostname(校验格式、自动识别通配符、同实例不重复),可选设置分组 / 备注 / 验证方式关联 |
| F4 删除域名 | 二次确认后移除域名;若仍被任一证书关联则**硬拦截** |
| F5 进入详情入口 | 逐条进入该域名的域名详情页 |

> 本页**不含**:批量新增 / 批量删除域名(modules/domains.md DECM1);域名维度统计卡片(总数 / 即将到期数归 dashboard);证书生命周期动作(归 certificates);验证方式配置与执行(归 acme)。

| 触发 | 去向 / 结果 | 说明 |
| --- | --- | --- |

## 4. 交互说明

- **列表范围与呈现**:列出全部域名;每条显示 hostname、通配符标识、分组、备注摘要、证书态投影、关联证书数。
- **证书态投影(聚合口径)**:证书态投影派生自 certificates 证书状态机(引用不复述)。一个域名可被多张证书关联,列表按**最紧急**的关联证书态呈现,优先级为:**失败(已过期 / 签发失败 / 续签失败)> 即将到期 > 有效**;该域名无任何关联证书时显示「无证书」。
- **筛选**:分组 / 证书态 / hostname 关键字三类,可单独或组合使用。
- **新增域名**:录入 hostname,校验格式;据 `*.` 前缀自动识别为**通配符域名**;校验同一实例内 hostname 不重复;可选设置分组、备注、验证方式关联(通配符域名的验证方式限 DNS-01)。
- **删除域名**:二次确认;若该域名仍被**任一现存证书**关联,则**拦截删除**并提示「该域名仍被 N 张证书关联,请先在证书管理处理相关证书」(flows/domains.md §3.3);未被任何证书关联时移除该域名。本页不代为删改证书。
- **无批量与统计**:不做批量操作、不放统计卡片。

---

## 5. 验收要点

> 业务口径,每条可判真伪。

1. 列表展示全部域名,每条显示 hostname、是否通配符、分组、备注摘要、证书态投影、关联证书数。
2. 域名的证书态投影取其关联证书中**最紧急**状态呈现,优先级为 失败 > 即将到期 > 有效;无任何关联证书时显示「无证书」。
3. 可按分组筛选。
4. 可按证书态筛选。
5. 可按 hostname 关键字筛选。
6. 新增域名录入 hostname 时校验格式,并据 `*.` 前缀自动识别为通配符域名。
7. 同一实例内 hostname 已存在时,新增被拒。
8. 新增域名可选设置分组 / 备注 / 验证方式关联;通配符域名的验证方式限 DNS-01。
9. 新增成功的域名初始证书态为「无证书」。
10. 删除域名需二次确认。
11. 被任一现存证书关联的域名删除被拦截,提示先在证书管理处理相关证书。
12. 未被任何证书关联的域名删除成功。
13. 点击某域名条目进入其域名详情页。
14. 本页不含批量新增 / 批量删除域名操作。
15. 本页不含域名维度统计卡片(统计归总览页)。
